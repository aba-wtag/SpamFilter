# SpamFilter

## How to run

```
python -m venv env
source env/bin/activate
pip install -r requirements.txt
```

### To run train the model

```
python pipeline.py
```

### To run the server

```
python app.py
```

### To run the client just open the `./client.spamfilter.html` in any browser

### To run the CLI

```
python predict_spam.py {your message}
```
